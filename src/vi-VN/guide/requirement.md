## Cấu hình máy

Cấu hình tối thiểu là mức cần thiết để chạy được game. Cấu hình khuyến nghị giúp bạn có trải nghiệm chơi game mượt mà.

#### Cấu hình tối thiểu:

| Yêu cầu | Thông số |
| ------- | -------- |
| CPU | Intel Core i3-9100 / AMD Ryzen 5 1400 |
| RAM | 12GB RAM |
| GPU | Nvidia GTX 950 / AMD Radeon R7 370 hoặc tương đương |
| Ổ cứng | 2GB+ |
| DirectX | DirectX 11 |
| Độ phân giải | 1280x720 |

> [!tip]
> Nếu máy bạn yếu hơn cấu hình tối thiểu, thử dùng phiên bản web với trình duyệt Chrome - có thể chạy được!

#### Cấu hình khuyến nghị:

| Yêu cầu | Thông số |
| ------- | -------- |
| CPU | Intel Core i5-9400 / AMD Ryzen 5 3600 trở lên |
| RAM | 16GB RAM trở lên |
| GPU | Nvidia GTX 1660 Super / AMD Radeon RX 590 trở lên |
| Ổ cứng | 3GB+ |
| DirectX | DirectX 12 |
| Độ phân giải | 1920x1080 |

## Yêu cầu hệ điều hành (Client)

| Yêu cầu | Thông số |
| ------ | -------- |
| **Phiên bản tối thiểu** | Windows 10 22H2 trở lên |
| **Phiên bản khuyến nghị** | Windows 11 23H2 trở lên |
| **Thành phần bắt buộc** | Windows Update |
| Thành phần tùy chọn | WebView2 Runtime |

> [!warning]
> Nếu game báo lỗi thiếu WebView2, tải từ [đây](https://developer.microsoft.com/en-us/microsoft-edge/webview2/).

## Yêu cầu chơi Online

Chúng tôi khuyến nghị dùng các trình duyệt sau để chơi phiên bản web:

| Trình duyệt | Ghi chú |
| ----------- | ------- |
| Chrome | Khuyến nghị Chrome 114 trở lên - chạy mượt nhất |
| Microsoft Edge | Có sẵn trên Windows 10/11 |
| Firefox | Có thể gặp một số vấn đề tương thích |

> [!info]
> **Mẹo hiệu năng:**
> - Đóng các tab không cần thiết
> - Tắt extension nặng
> - Dùng chế độ Hardware Acceleration
