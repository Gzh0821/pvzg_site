## Impzane Edition

Impzane Edition is a community mod for PvZ2 Gardendless by Seal-enjoyer. Its README presents it as a harder and more chaotic remix that changes plants and zombies across the game.

## Features

- Reworks every plant and zombie for a more chaotic and harder experience

- Includes level and world-map progression changes

- Provides an author-maintained plant changelog for detailed balance changes

- The author describes an update schedule of about every 1.5 months, with each update adding levels and balance changes

## Note

Download the GitHub repository zip and load it through the F10 `Datapacks` panel as described in the author's README.

## Download

You can download the mod from [GitHub](https://github.com/Seal-enjoyer/PvZ-GE-Impzane-edition).
