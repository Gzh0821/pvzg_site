> [!tip]
> Starting from version `0.7.1`, PvZ2 Gardendless has integrated a powerful new modding tool called `GP-Next`. It offers enhanced features and a more user-friendly experience compared to the older `gePatcher`. If you're using version `0.7.1` or above, we highly recommend using `GP-Next` for your modding needs. For users of earlier versions, the `gePatcher` tutorial is still available and can be found [here](patcher.md).
>
> The development team does not guarantee that this feature will be supported in future versions, which depends on the game's updates and maintenance.
