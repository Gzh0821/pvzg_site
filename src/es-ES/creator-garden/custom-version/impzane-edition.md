## Impzane Edition

Impzane Edition es un mod de la comunidad para PvZ2 Gardendless creado por Seal-enjoyer. Su README lo presenta como una version mas caotica y dificil que cambia plantas y zombis en todo el juego.

## Features

- Reajusta todas las plantas y zombis para una experiencia mas caotica y dificil

- Incluye cambios de niveles y progreso del mapa

- Incluye un registro de cambios de plantas mantenido por el autor

- El autor indica un calendario de actualizacion aproximado de cada 1,5 meses, con nuevos niveles y cambios de balance en cada actualizacion

## Note

Descarga el zip del repositorio de GitHub y cargalo desde el panel F9 en `Datapacks`, como indica el README del autor.
