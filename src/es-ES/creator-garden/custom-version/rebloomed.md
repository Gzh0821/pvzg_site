## Sandy's Pvz2: Rebloomed

Sandy's Pvz2: Rebloomed es un mod de campana para PvZ2 Gardendless creado por Sandy. Su `pack.json` lo describe como una gran renovacion de PvZ2 Gardendless.

## Features

- Rehace el progreso de la campana; la version actual llega hasta Pirate Seas Day 5

- El README marca la dificultad vanilla como C y lista rutas mas dificiles para jugadores que ya completaron el PvZ2 original

- Incluye cambios en el orden de mundos, empezando por Players House y Ancient Egypt antes de avanzar a mundos posteriores

## Note

`pack.json` indica que esta pensado para la version `0.7.1` del juego y requiere GP-Next `>=1.0.0`.

## Download

Puedes descargar el mod desde [GitHub](https://github.com/SandyBrawlStars/Pvz2Rebloom).
