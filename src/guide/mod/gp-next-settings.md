# 设置与扩展

Settings 页除了常规开关，还承载了一部分由 GP-Next 接管的运行时能力。

## 基础设置

目前常见的基础设置包括：

- 面板语言
- 帧率
- 调试模式

## 滚动优化

这是最近 GP-Next 接管的一类运行时功能。

目前它主要分两部分：

### 1. 连续滚动灵敏度

对应类似这些场景：

- 设置页滚动
- 图鉴/类似滚动页面
- 一些连续 wheel / scroll 场景

### 2. 离散选择栏最小间隔

对应类似这些场景：

- 世界选择器
- 沙盒植物栏
- 沙盒僵尸栏
- 部分沙盒滚轮切换入口

这个设置的目标不是“滚得更快”，而是减少触控板或高灵敏度输入带来的误触和连跳。

## Runtime Extensions

当前最重要的是：

- `Dynamic Plant Registry`

它的作用是：

- 接管一部分植物身份映射逻辑
- 让“新增植物 / 克隆植物”的数据包在运行时更稳定

### 什么时候要开它

如果你的 Datapack：

- 新增了植物
- 克隆了现有植物
- 修改了植物身份与映射相关逻辑

那通常就应该保持开启。

### 生效方式

这类设置通常：

- **重新加载补丁即可生效**
- 不一定必须重启游戏

## HP Overlay

这是战斗内的血量显示功能。

目前可以显示：

- 植物血量
- 僵尸血量
- Tomb 类地物血量

对于带额外承伤层的对象，还会显示主血量之外的第二层信息。

## About、Log 与更新检查

### About

用于查看 GP-Next 简介和控制台命令说明。

### Log

用于查看 GP-Next 运行日志，排查：

- 补丁有没有加载
- 哪一步失败了
- 某个运行时扩展有没有生效

### 页脚更新检查

页脚会显示：

- 当前 GP-Next 版本
- 是否检查到新版本

检查到更新后，可以直接跳转到官网下载页。
