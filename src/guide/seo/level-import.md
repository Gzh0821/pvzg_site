## 场景说明

你可以从创意花园下载社区/官方关卡 JSON，然后在游戏内通过 `Play Local Level` 导入游玩。

## 导入步骤

1. 打开 [创意花园](../../creator-garden/)
2. 进入官方关卡或社区关卡列表
3. 点击关卡标题下载 JSON 文件
4. 启动游戏并进入设置菜单
5. 点击 `Play Local Level`
6. 选择刚下载的关卡 JSON 文件并确认

## 导入失败时排查

### 1. 文件无法识别

- 确认是 JSON 文件
- 确认文件未被网盘二次包装或重命名
- 尝试重新下载

### 2. 进入关卡后异常

- 检查关卡是否对应当前游戏版本
- 查看作者备注中的版本要求
- 尝试使用最新版客户端

### 3. 导入后无反应

- 重启游戏后重试
- 确认路径权限允许读取
