## Sandy's Pvz2: Rebloomed

Sandy's Pvz2: Rebloomed 是 Sandy 制作的 PvZ2 Gardendless 战役改版。`pack.json` 将它描述为一个大型 Gardendless 重制 Mod。

## Features

- 重新设计战役流程，当前版本进度到 Pirate Seas Day 5

- README 标注原版难度为 C，并提供更高难度路线给有原版 PvZ2 经验的玩家挑战

- 包含世界顺序调整，从 Players House、Ancient Egypt 到后续世界逐步推进

## Note

`pack.json` 标注适用于游戏版本 `0.7.1`，并要求 GP-Next `>=1.0.0`。

## Download

You can download the mod from [GitHub](https://github.com/SandyBrawlStars/Pvz2Rebloom).
