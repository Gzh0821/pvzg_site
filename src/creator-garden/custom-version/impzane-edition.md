## Impzane Edition

Impzane Edition 是 Seal-enjoyer 为 PvZ2 Gardendless 制作的社区 Mod。仓库 README 将它定位为一个更混乱、更困难的改版，会重新调整植物和僵尸的表现。

## Features

- 所有植物和僵尸都有重新调整，整体体验更混乱、难度更高

- 包含关卡与世界地图进度改动

- 作者维护了植物改动记录，方便查看具体平衡变化

- 作者说明更新节奏约为每 1.5 个月，每次更新会加入一批关卡和平衡调整

## Note

下载 GitHub 仓库 zip 后，可按作者 README 的说明通过 F9 面板的 `Datapacks` 加载。
