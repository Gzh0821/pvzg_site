## Sandy's Pvz2: Rebloomed

Sandy's Pvz2: Rebloomed - это мод кампании для PvZ2 Gardendless от Sandy. В `pack.json` он описан как крупная переработка PvZ2 Gardendless.

## Features

- Перерабатывает прохождение кампании; текущая версия доходит до Pirate Seas Day 5

- README отмечает сложность vanilla как C и указывает более сложные маршруты для игроков, уже проходивших оригинальную PvZ2

- Включает изменения порядка миров, начиная с Players House и Ancient Egypt перед переходом к более поздним мирам

## Note

`pack.json` указывает версию игры `0.7.1` и требует GP-Next `>=1.0.0`.
