## MadCreeper's Balance

MadCreeper's Balance is a Gardendless balance mod by MadCreeper7611. It focuses on broad balance adjustments across plants, zombies, tiles, and selected levels.

## Features

- Rebalances many plants with buffs and nerfs to sun cost, recharge, damage, freezing/heating behavior, and plant food effects

- Adjusts many zombies and bosses, including health, speed, wave point costs, spawn patterns, and special resistances

- Includes selected tile and level changes, such as gold tile sun timing and revised conveyor or starting conditions in several levels

## Note

The repository README is maintained as the changelog. Review it before installing, because the mod changes many balance numbers across the game.

The GitHub repository contains a `MadCreeper's Balance/pack.json` and `jsons/` directory, so it is packaged as a GP-Next datapack-style mod.

## Download

You can download the mod from [GitHub](https://github.com/madcreeper7611/MadCreepers-Balance/).
