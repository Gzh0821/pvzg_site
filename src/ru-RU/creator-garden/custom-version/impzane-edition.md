## Impzane Edition

Impzane Edition - это мод сообщества для PvZ2 Gardendless от Seal-enjoyer. В README он описан как более хаотичная и сложная переработка, меняющая растения и зомби во всей игре.

## Features

- Перерабатывает все растения и зомби для более хаотичного и сложного прохождения

- Включает изменения уровней и прогресса на карте мира

- Автор ведет отдельный журнал изменений растений

- Автор указывает примерный график обновлений раз в 1,5 месяца: новые уровни и изменения баланса в каждом обновлении

## Note

Скачайте zip из репозитория GitHub и загрузите его через панель F9 в `Datapacks`, как указано в README автора.
