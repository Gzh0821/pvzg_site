# GP-Next

GP-Next - основной инструмент для моддинга и отладки в PvZ2 Gardendless. С его помощью можно:

- управлять datapack-пакетами и одиночными патчами
- просматривать и редактировать игровые данные во время работы игры
- создавать языковые пакеты и патчи уровней
- использовать функции времени выполнения, такие как Trainer, HP Overlay и Runtime Extensions
- вызывать функции `window.gpNext` из консоли
