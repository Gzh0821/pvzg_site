安装与启动常见报错处理

适用范围

  • Windows 桌面版
  • macOS 版本（Apple Silicon）
  • Linux Docker 本地部署版本

Windows 常见问题

1. 启动后闪退或无反应

  1. 检查系统是否满足最低配置
  2. 确保显卡驱动与系统补丁为较新版本
  3. 删除旧版本后重新下载最新版并解压到纯英文路径
  4. 尝试关闭安全软件实时拦截后再次运行

2. 提示缺少运行库

  1. 安装或修复 Microsoft Visual C++ 运行库
  2. 重启系统后再次打开游戏

3. 存档不兼容

  1. 官网版本与网盘版本打包方式不同
  2. 建议在同一渠道内完成导出与导入
  3. 迁移前请先备份原存档

macOS 常见问题

1. 提示“已损坏，无法打开”

可在终端执行：

sudo xattr -r -d com.apple.quarantine <应用路径>

然后在“系统设置 > 隐私与安全性”中允许应用。

Linux Docker 常见问题

1. 容器启动成功但无法访问

  1. 检查端口映射是否正确
  2. 检查本机防火墙与代理设置
  3. 确认镜像版本与文档一致
