Configuration

The minimum configuration requirement is the minimum configuration to run the game, and the recommended configuration is to get a standard gaming experience.

Minimum Configuration：

RequirementSpecification
ProcessorIntel Core i3-6100 / AMD Ryzen 3 1200
RAM8GB RAM
GPUNvidia GTX 950 / AMD Radeon R7 370 or equivalent graphics card
Storage1GB+
DirectXDirectX 11
Resolution1280x720

Recommended Configuration:

RequirementSpecification
ProcessorIntel Core i5-9400 / AMD Ryzen 5 3600 or higher performance
RAM16GB RAM or higher
GPUNvidia GTX 1660 Super / AMD Radeon RX 590 or higher performance
Storage2GB+
DirectXDirectX 12
Resolution1920x1080
